## نکته‌ها

<Callout type="tip" title="نکته">
اگر از کلودفلر سرتیفیکیت دریافت کردید بخش `ocspStapling` را از اینباند خود حذف کنید.
</Callout>

<Callout type="tip" title="نکته">
اگر قصد استفاده از `2` یا چند دامین و یا ساب دامین را دارید می‌توانید مثل نمونه زیر چندین سرتیفیکت را داخل اینباند قرار بدید.

```json
            "certificates": [
              {
                "ocspStapling": 3600,
                "certificateFile": "/var/lib/pasarguard/certs/domain1.com/fullchain.pem",
                "keyFile": "/var/lib/pasarguard/certs/domain1.com/key.pem"
              },
              {
                "ocspStapling": 3600,
                "certificateFile": "/var/lib/pasarguard/certs/domain2.com/fullchain.pem",
                "keyFile": "/var/lib/pasarguard/certs/domain2.com/key.pem"
              }
            ],
```
</Callout>

<Callout type="tip" title="نکته">
اگر از `Fallback` استفاده می‌کنید، لازم هست تا در تنظیمات `Core` موردنظرتون تگ اینباند فالبک خود را انتخاب کنید.
</Callout>
