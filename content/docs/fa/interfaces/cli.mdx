# خط فرمان PasarGuard

CLI همراه PasarGuard نصب می‌شود و در حال حاضر برای ساخت کلیدهای راه‌اندازی و دستورهای پایه نگهداری استفاده می‌شود.

## دستورها

```bash
# نمایش راهنما
pasarguard cli --help

# نمایش نسخه نصب‌شده
pasarguard cli version

# ساخت کلید موقت یک‌بارمصرف برای راه‌اندازی
pasarguard cli generate-temp-key
```

## کلید راه‌اندازی مالک

قبل از باز کردن فرم **Owner access** در داشبورد، دستور `generate-temp-key` را اجرا کنید:

```bash
pasarguard cli generate-temp-key
```

این دستور یک کلید موقت و زمان انقضای آن را چاپ می‌کند. کلید فقط 5 دقیقه معتبر است و فقط یک‌بار قابل استفاده است.

فرم راه‌اندازی داشبورد می‌تواند با این کلید اولین حساب مالک را بسازد، رمز مالک را بازنشانی کند، مالک را حذف کند یا یک ادمین موجود را به مالک ارتقا دهد.
