# رابط‌های PasarGuard

PasarGuard یک رابط خط فرمان کوچک برای کارهای راه‌اندازی و نگهداری دارد.

## ابزار موجود

<Cards>
  <Card title="راهنمای CLI" href="/interfaces/cli">
    ساخت کلیدهای موقت راه‌اندازی، بررسی نسخه نصب‌شده و دیدن راهنمای دستورات.
  </Card>
</Cards>

## شروع سریع

```bash
# نمایش دستورهای موجود
pasarguard cli --help

# ساخت کلید موقت برای راه‌اندازی مالک
pasarguard cli generate-temp-key
```

از کلید ساخته‌شده در فرم **Owner access** داشبورد برای ساخت، بازنشانی، حذف یا ارتقای حساب مالک استفاده کنید.
