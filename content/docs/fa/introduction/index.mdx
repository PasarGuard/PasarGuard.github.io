<h2>PasarGuard، راه‌حل یکپارچه GUI مقاوم در برابر سانسور</h2>

PasarGuard ابزاری برای مدیریت پروکسی است که رابط کاربری ساده و آسانی برای مدیریت صدها حساب پروکسی فراهم می‌کند که با Xray-core قدرت گرفته و با استفاده از Python و Reactjs ساخته شده است.

<h3>چرا از PasarGuard استفاده کنیم؟</h3>

PasarGuard کاربرپسند، غنی از ویژگی و قابل اعتماد است. این ابزار به شما امکان ایجاد پروکسی‌های مختلف برای کاربرانتان بدون هیچ پیکربندی پیچیده‌ای را می‌دهد. با استفاده از رابط وب داخلی آن، می‌توانید کاربران را نظارت، تغییر و محدود کنید.

<h3>ویژگی‌ها</h3>

<ul>
  <li>رابط وب داخلی</li>
  <li>بک‌اند REST API کامل</li>
  <li>پشتیبانی از چندین گره (برای توزیع زیرساخت و مقیاس‌پذیری)</li>
  <li>پشتیبانی از پروتکل‌های Vmess، VLESS، Trojan و Shadowsocks</li>
  <li>چندپروتکلی برای یک کاربر</li>
  <li>چندکاربری روی یک ورودی</li>
  <li>چندین ورودی روی یک پورت (پشتیبانی از fallback)</li>
  <li>محدودیت‌های ترافیک و تاریخ انقضا</li>
  <li>محدودیت ترافیک دوره‌ای (مثل روزانه، هفتگی و غیره)</li>
  <li>لینک اشتراک‌گذاری سازگار با V2ray (مثل V2RayNG، SingBox، Nekoray و غیره)، Clash و ClashMeta</li>
  <li>تولیدکننده خودکار لینک اشتراک و QR کد</li>
  <li>نظارت سیستم و آمار ترافیک</li>
  <li>پیکربندی قابل تنظیم xray</li>
  <li>پشتیبانی از TLS و REALITY</li>
  <li>ربات تلگرام یکپارچه</li>
  <li>رابط خط فرمان یکپارچه (CLI)</li>
  <li>چندزبانه</li>
  <li>پشتیبانی از چندین ادمین (در حال توسعه)</li>
</ul>
