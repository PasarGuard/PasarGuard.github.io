# CLI PasarGuard

CLI устанавливается вместе с PasarGuard и сейчас используется для ключей настройки и базовых команд обслуживания.

## Команды

```bash
# Показать справку
pasarguard cli --help

# Показать установленную версию
pasarguard cli version

# Создать одноразовый временный ключ настройки
pasarguard cli generate-temp-key
```

## Ключ настройки владельца

Выполните `generate-temp-key` перед открытием формы **Owner access** в панели:

```bash
pasarguard cli generate-temp-key
```

Команда выводит временный ключ и время его истечения. Ключ действует 5 минут и может быть использован только один раз.

Форма настройки панели может использовать этот ключ для создания первой учетной записи владельца, сброса пароля владельца, удаления владельца или повышения существующего администратора до владельца.
