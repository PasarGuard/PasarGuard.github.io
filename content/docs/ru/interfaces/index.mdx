# Интерфейсы PasarGuard

PasarGuard включает небольшой интерфейс командной строки для задач настройки и обслуживания.

## Доступный инструмент

<Cards>
  <Card title="Руководство по CLI" href="/interfaces/cli">
    Создание одноразовых ключей настройки, проверка установленной версии и просмотр справки по командам.
  </Card>
</Cards>

## Быстрый старт

```bash
# Показать доступные команды
pasarguard cli --help

# Создать одноразовый ключ настройки владельца
pasarguard cli generate-temp-key
```

Используйте созданный ключ в форме **Owner access** панели, чтобы создать, сбросить, удалить или повысить учетную запись владельца.
