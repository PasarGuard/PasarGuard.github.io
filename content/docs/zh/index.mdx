<h2>PasarGuard，统一的反审查GUI解决方案</h2>

PasarGuard是一个代理管理工具，提供简单易用的用户界面来管理数百个代理账户，基于Xray-core构建，使用Python和Reactjs开发。

<h3>为什么使用PasarGuard？</h3>

PasarGuard用户友好、功能丰富且可靠。它让您无需复杂配置即可为用户创建不同的代理。使用其内置的Web UI，您可以监控、修改和限制用户。

<h3>功能特性</h3>

<ul>
  <li>内置Web UI</li>
  <li>完整的REST API后端</li>
  <li>多节点支持（用于基础设施分布和可扩展性）</li>
  <li>支持Vmess、VLESS、Trojan和Shadowsocks协议</li>
  <li>单用户多协议</li>
  <li>单入站多用户</li>
  <li>单端口多入站（支持回退）</li>
  <li>流量和到期日期限制</li>
  <li>周期性流量限制（如每日、每周等）</li>
  <li>与V2ray（如V2RayNG、SingBox、Nekoray等）、Clash和ClashMeta兼容的订阅链接</li>
  <li>自动分享链接和二维码生成器</li>
  <li>系统监控和流量统计</li>
  <li>可自定义的xray配置</li>
  <li>TLS和REALITY支持</li>
  <li>集成Telegram机器人</li>
  <li>集成命令行界面（CLI）</li>
  <li>多语言支持</li>
  <li>多管理员支持（开发中）</li>
</ul>
