# PasarGuard 接口

现代化的命令行和文本用户界面工具，用于高效且优雅地管理 PasarGuard。

## 概述

PasarGuard 提供两个强大的接口选项来管理您的代理基础设施：

- **CLI（命令行界面）** - 使用 Typer 构建的现代、安全的命令行工具
- **TUI（文本用户界面）** - 使用 Textual 构建的交互式、可视化终端界面

这两种工具都提供全面的管理员管理、系统监控和高效操作，无需网络浏览器。

## 为什么使用 PasarGuard 接口？

PasarGuard 接口专为效率和灵活性而设计：

- **快速访问** - 直接从终端管理系统
- **安全操作** - 具有确认功能的交互式提示以确保安全
- **美观输出** - 丰富的格式、表格和可视化反馈
- **无需浏览器** - 无需网络访问即可实现完整功能
- **自动化就绪** - 非常适合脚本和自动化工作流

## 主要功能

### CLI 功能

- 🎯 **安全的 CLI** 具有精美的输出格式
- 📊 **精美的表格和面板** 用于数据显示
- 🔒 **安全的管理员管理** 具有交互式提示
- 📈 **系统状态监控** 和问题检测
- ⌨️ **交互式提示** 具有确认功能以增强安全性

### TUI 功能

- 🎯 **交互式 TUI** 支持鼠标和键盘操作
- 📱 **响应式设计** 支持暗色模式
- ⌨️ **键盘快捷键** 用于快速导航
- 🔄 **实时更新** 和数据实时刷新
- 📊 **可视化数据显示** 具有颜色和格式化
- 🔒 **安全的管理员管理** 具有可视化反馈

## 开始使用

要开始使用 PasarGuard 接口，您可以直接访问它们：

```bash
# 访问 CLI
pasarguard cli --help

# 访问 TUI
pasarguard tui
```

这两种工具都包含在 PasarGuard 安装中，可以立即使用。

## 使用场景

- **服务器管理** - 管理管理员和监控系统状态
- **快速操作** - 快速创建、修改和删除管理员
- **远程访问** - 通过 SSH 完全控制，无需 Web 界面
- **自动化** - 集成到脚本和自动化工作流中
- **故障排除** - 快速诊断和系统信息
