# PasarGuard 接口

PasarGuard 包含一个用于设置和维护任务的小型命令行接口。

## 可用工具

<Cards>
  <Card title="CLI 指南" href="/interfaces/cli">
    生成一次性设置密钥、检查已安装版本并查看命令帮助。
  </Card>
</Cards>

## 快速开始

```bash
# 显示可用命令
pasarguard cli --help

# 生成一次性所有者设置密钥
pasarguard cli generate-temp-key
```

在仪表板的 **Owner access** 表单中使用生成的密钥，以创建、重置、删除或提升所有者账号。
