# PasarGuard CLI

CLI 会随 PasarGuard 一起安装，目前用于设置密钥和基础维护命令。

## 命令

```bash
# 显示帮助
pasarguard cli --help

# 显示已安装版本
pasarguard cli version

# 生成一次性临时设置密钥
pasarguard cli generate-temp-key
```

## 所有者设置密钥

打开仪表板 **Owner access** 表单前，请先运行 `generate-temp-key`：

```bash
pasarguard cli generate-temp-key
```

该命令会输出临时密钥及其过期时间。密钥有效期为 5 分钟，并且只能使用一次。

仪表板设置表单可以使用此密钥创建第一个所有者账号、重置所有者密码、删除所有者，或将现有管理员提升为所有者。
